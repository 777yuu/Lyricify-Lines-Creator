# Lyricify Lines Creator
Lyricify Lines 逐行歌词打轴工具。是 Lyricify Lines 歌词的制作工具，同时也支持 LRC 歌词的输出。

## 技术
使用 C++ 开发，并使用 HiEasyX 图形库。

## 特点
1. 操作简便，功能完善。
2. 可标记行结束时间。
3. 支持高 DPI。
4. 支持多语言本地化。

## 开发原因
Lyricify 系列缺少 Lyricify Lines 歌词的打轴工具。
